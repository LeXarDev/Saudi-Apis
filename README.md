## 🚀 نظرة عامة

**منصة واجهات البرمجة والبيانات المفتوحة** هي مرجعك الشامل في المملكة العربية السعودية. نهدف إلى:
- تسهيل وصول المطورين للخدمات والبيانات المفتوحة
- توفير مصدر موحد للبيانات الحكومية والخاصة
- دعم المطورين والباحثين في الوصول للمعلومات

## 📊 المحتوى المتوفر

### 🔌 واجهات البرمجة (APIs)
- واجهات برمجية حكومية رسمية
- خدمات البيانات المباشرة
- توثيق تفصيلي لكل واجهة
- أمثلة عملية للاستخدام

### 💾 البيانات المفتوحة
- مجموعات البيانات الحكومية
- إحصائيات رسمية ومعتمدة
- بيانات جغرافية ومكانية
- معلومات اقتصادية وتعليمية

## ⚡ المميزات

- 🔍 **بحث متقدم**: محرك بحث ذكي للواجهات والبيانات
- 📱 **تصميم متجاوب**: يعمل بشكل مثالي على جميع الأجهزة
- 🌙 **الوضع الليلي**: تجربة مريحة للعين
- 🔄 **تحديثات مستمرة**: إضافة مصادر جديدة بشكل دوري
- 📖 **توثيق شامل**: شرح مفصل لكل مصدر
- 🌐 **دعم كامل للعربية**: واجهة مستخدم ثنائية اللغة

## 💻 البدء السريع

1. **استنساخ المشروع**
   ```bash
   git clone https://github.com/LeXarDev/Saudi-Apis.git
   ```

2. **تثبيت المتطلبات**
   ```bash
   cd Saudi-Apis
   npm install
   ```

3. **تشغيل المشروع**
   ```bash
   npm run dev
   ```
